# 💌 Carta para alguien especial

Una experiencia web interactiva tipo carta vintage, diseñada para transmitir un mensaje sincero, delicado y emocional.

## ✨ Descripción

Este proyecto es una carta digital con estética romántica y clásica, creada con HTML, CSS y JavaScript puro.
Incluye animaciones suaves, música de fondo y una presentación cuidada para generar una experiencia íntima y especial.

## 🎨 Características

- Estilo vintage / romántico
- Tipografía manuscrita
- Animación de apertura
- Efecto máquina de escribir
- Música de fondo
- Corazones flotantes
- Diseño responsive
- Favicon personalizado
